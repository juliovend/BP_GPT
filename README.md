# Bruit Parasite — Site vitrine statique

## Structure
- `index.html`
- `spectacles.html`
- `ateliers.html`
- `css/style.css`
- `js/script.js`
- `assets/images/`

## Organisation des contenus
- La page d’accueil (`index.html`) affiche une sélection courte : **3 spectacles** et **3 ateliers**.
- La page `spectacles.html` affiche la liste complète des spectacles.
- La page `ateliers.html` affiche la liste complète des ateliers.

## Ajouter un nouveau spectacle
1. Ouvrir `spectacles.html`.
2. Dupliquer une carte de spectacle existante.
3. Modifier le titre, le label, la description, les infos pratiques et l’image.
4. Si aucune image n’existe, utiliser le placeholder CSS pour éviter toute image cassée.

## Ajouter un nouvel atelier
1. Ouvrir `ateliers.html`.
2. Dupliquer une carte d’atelier existante.
3. Modifier le titre, le sous-titre, la description, les infos pratiques et l’image.
4. Si aucune image n’existe, utiliser le placeholder CSS.

## Mise en ligne sur Hostinger
Déposer dans `public_html` :
- `index.html`
- `spectacles.html`
- `ateliers.html`
- `css/`
- `js/`
- `assets/`
